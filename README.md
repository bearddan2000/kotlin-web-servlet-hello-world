# kotlin-web-servlet-hello-world

## Description
A POC for a kotlin jsp project.

## Tech stack
- kotlin

## Docker stack
- gradle:4.2.1-jdk8-alpine

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
